# Multi-step Form
A multi-step form with persistent data on submit, validation on change, links to each step, and responsiveness. 

### Features
- Data is saved to local storage on submit
- Form validation on change to avoid skipping steps through next button or sidebar link
- Responsiveness to different screen sizes

### Built With
- React + Vite
- React Hook Form
- React Router
- Tailwind CSS
- Radix UI
- Zustand

### Getting Started
In order to setup and work on this project on your own, you will need to:

Clone this project:<br>
`git clone https://github.com/nofuenterr/multi-step-form.git`

Once you have cloned this project, you can install the required dependencies by using:<br>
`npm install`

A live demo of the project can be started by using:<br>
`npm run preview`

Distribution files can be produced using:<br>
`npm run build`
